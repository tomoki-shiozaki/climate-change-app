# 気候変動データ閲覧アプリ

Django REST Framework（API）と React（UI）を使った、気候変動データの可視化アプリです。

## 使用技術

- Django REST Framework（JWT 認証）
- React
- PostgreSQL

## 機能

- 気温データのグラフ表示
- JWT による認証・ログイン/ログアウト

## デプロイ URL
