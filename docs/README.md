# 気候変動データ可視化アプリ開発プロジェクト

## 1. プロジェクト概要

- **プロジェクト名：** 気候変動データ可視化アプリ開発プロジェクト
- **前提：** 世界各国の気候変動関連データ（気温変化、CO₂ 濃度、海面上昇など）が API で提供されている。個人開発でも利用可能なオープンデータとして、Our World in Data を利用する。
  - データ例
    - 世界平均気温の変化（年次）
    - 二酸化炭素濃度（年次）
    - 国・地域ごとの気候指標
- **開発目的：**
  - React+DRF でオリジナルアプリを作る
  - 自分自身が、気候変動について理解を深める
  - ユーザー（他の人）にとっても、気候変動に関して理解を深める一助となるアプリとする
  - インタラクティブ性を盛り込む
- **開発目的：** 気候変動データを可視化し、ユーザーが世界の変化を直感的に理解できるアプリを作る。
  - MVP ではグラフ表示＋指標切替＋地域選択を中心に実装
  - 将来的には比較機能や未来予測も追加可能
- **MVP**
  - 1 ～ 2 指標（気温アノマリー・CO₂ 濃度）
  - 簡単なグラフ
  - 文章解説

## 2. ターゲットユーザー

- 気候変動に関心のある一般ユーザーや学習者
- 自分の興味に応じてデータを比較・観察したい人

## 3. 利用シーンの想定

- 世界全体または特定地域の気温変化をグラフで確認
- CO₂ 濃度の推移を年次で確認
- 指標を切り替えて比較表示
- （拡張）地域ごとの未来予測やランキング表示
- （拡張）特定の期間や地域をお気に入り登録して追跡
